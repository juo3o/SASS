sass 믹싱기법(mixin, include)

1. $mixin 문법

웹사이트 전체에 재사용할 css코드를 정의하는 문

        @mixin mixin명{
            //반복해서 사용할 스타일 서식을 작성한다
        }

2. @include 문법

@mixin을 선택자에 포함할 때 사용하는 지시문

        선택자{
            @include mixin명;
        }

3. @mixin을 중첩하여 새로운 @mixin만들기

        @mixin 믹스인명A {
            //재사용할 스타일 서식을 작성
        }
        @mixin 믹스인명B {
            //재사용할 스타일 서식을 작성
        }
        @mixin 믹스인명C {
            @include 믹스인명A;
            @include 믹스인명B;
        }
